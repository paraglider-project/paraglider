.. _roadmap:

Roadmap
--------------

This pages lists some of the features that are planned for the future of Paraglider.

Features:
 * Get the IBM plugin caught up with the other plugins in feature completeness. :bdg-warning:`Short-term`
 * Add support for additional resource types (e.g. specialized IaaS resources, PaaS resources, etc.). :bdg-primary:`Longer-term`
 * Support for brownfield scenarios like interop, or reverse engineering. :bdg-primary:`Longer-term`
 * Add support for network functions such as load balancers, firewalls, etc. :bdg-primary:`Longer-term`
 * Support for assigning public/static IP addresses. :bdg-secondary:`As Needed`
 * Add plugins for other clouds. :bdg-secondary:`As Needed`

Technical:
 * Support asynchronous requests to the Paraglider controller. :bdg-primary:`Longer-term`
 * Improve provisioning latency for inter-cloud connectivity. :bdg-primary:`Longer-term`

Integration:
 * Integrate with k8s networking plugins. :bdg-primary:`Longer-term`
 * Infrastructure-as-code support (e.g., Terraform). :bdg-primary:`Longer-term`
